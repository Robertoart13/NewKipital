# ⚙️ Playbook de Incidentes

## 🎯 Severidades
- SEV1: caída total o riesgo legal crítico.
- SEV2: degradación importante de operación.
- SEV3: impacto parcial con workaround.
- SEV4: impacto menor.

## 🔄 Flujo
1. Detección.
2. Clasificación severidad.
3. Asignación Incident Commander.
4. Contención.
5. Mitigación.
6. Comunicación.
7. Cierre y postmortem.

## 🎯 Tiempos objetivo
- SEV1: respuesta inicial <= 15 min.
- SEV2: respuesta inicial <= 30 min.
- SEV3: respuesta inicial <= 2 h.
- SEV4: respuesta inicial <= 1 día hábil.
