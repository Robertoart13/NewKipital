# ⚙️ SLO y KPI de Operacion

## 📊 SLO propuestos
| Indicador | Objetivo |
|---|---|
| Disponibilidad API | >= 99.9% mensual |
| Error rate 5xx | <= 0.5% |
| Latencia p95 endpoints críticos | <= 800ms |
| Éxito jobs worker | >= 99% |
| Cierre planilla sin rollback manual | >= 99% |

## 📊 KPI funcionales
| KPI | Formula |
|---|---|
| Tiempo ciclo planilla | fecha_aplicada - fecha_abierta |
| Acciones aprobadas a tiempo | aprobadas_en_SLA / total_aprobables |
| Incidentes por módulo | total_incidentes_modulo / periodo |

## 🎯 Revisión
- Semanal para operación.
- Mensual para comité técnico.
