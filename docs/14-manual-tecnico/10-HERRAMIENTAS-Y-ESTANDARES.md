# 🛠️ Manual Tecnico - Herramientas y Estandares

## 🎯 Herramientas base
- Node.js + NestJS + TypeORM
- React + Vite + TypeScript + Ant Design
- MySQL
- Redux Toolkit + TanStack Query

## 🎯 Estandares de implementacion
- Validacion server-side para toda entrada de usuario.
- Permisos backend obligatorios por endpoint.
- Guard clauses y errores con contexto.
- Auditoria en operaciones de impacto.
- No exponer datos sensibles sin permiso `employee:view-sensitive`.

## 🎯 Estandares de documentacion
- Manual de usuario para operacion diaria.
- Manual tecnico para ingenieria.
- Enlaces cruzados obligatorios entre documentos relacionados.
- Cambios de proceso deben actualizar ambos manuales.

## 🎯 Estandares de entrega
- Cambio funcional debe incluir:
  - ajuste de codigo
  - validacion QA
  - actualizacion documental
