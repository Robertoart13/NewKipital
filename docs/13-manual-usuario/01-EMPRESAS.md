# Manual de Usuario - Empresas

## Como crear una empresa
1. Ir a Configuracion > Empresas.
2. Click en Crear empresa.
3. Completar campos.
4. Confirmar.

## Campos y para que sirve cada uno
| Campo | Uso | Obligatorio |
|---|---|---|
| Nombre empresa | Identifica la empresa en todo el sistema | Si |
| Identificacion fiscal | Identificador legal/fiscal | Si |
| Moneda por defecto | Base de calculos y montos | Si |
| Estado | Activa o inactiva para operar | Si |
| Logo | Imagen institucional | No |
