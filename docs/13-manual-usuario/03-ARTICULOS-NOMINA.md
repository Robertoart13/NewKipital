# Manual de Usuario - Articulos de Nomina

## Como crear un articulo
1. Ir a Parametros de Planilla > Articulos de nomina.
2. Click en Crear articulo.
3. Completar campos.
4. Guardar.

## Campos y significado
| Campo | Uso | Obligatorio |
|---|---|---|
| Empresa | Empresa dueña del articulo | Si |
| Nombre articulo | Nombre operativo del concepto | Si |
| Tipo articulo | Naturaleza del concepto (ingreso/deduccion/etc.) | Si |
| Tipo accion personal | Relacion con acciones de personal | Si |
| Cuenta debito | Registro contable debito | Segun tipo |
| Cuenta credito | Registro contable credito | Segun tipo |
| Estado | Activo/Inactivo | Si |
