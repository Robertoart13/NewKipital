## Estandar de evidencia visual
- Formato recomendado: PNG.
- Resolucion minima: 1366x768.
- Nombre de archivo: `EV-XX-NOMBRE-PROCESO.png`.
- Cada figura debe incluir: modulo, proceso, paso y fecha.
- Carpeta recomendada: `docs/13-manual-usuario/evidencias/`.
