# Manual de Usuario - Cuentas Contables

## Como crear una cuenta
1. Ir a Configuracion > Cuentas contables.
2. Click en Crear cuenta.
3. Completar datos.
4. Guardar.

## Campos y significado
| Campo | Uso | Obligatorio |
|---|---|---|
| Empresa | Empresa propietaria de la cuenta | Si |
| Codigo cuenta | Codigo contable unico por empresa | Si |
| Nombre cuenta | Nombre de la cuenta | Si |
| Tipo cuenta | Clasificacion contable | Si |
| Permite movimiento | Define si acepta asientos/movimientos | Si |
| Estado | Activa/Inactiva | Si |
