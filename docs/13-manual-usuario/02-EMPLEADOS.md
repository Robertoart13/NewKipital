# Manual de Usuario - Empleados

## Como crear un empleado
1. Ir a Empleados.
2. Click en Crear empleado.
3. Completar datos personales y laborales.
4. Guardar.

## Campos y significado
| Campo | Uso | Obligatorio |
|---|---|---|
| Codigo empleado | Identificador interno unico | Si |
| Nombre completo | Identificacion de la persona | Si |
| Empresa | Empresa activa del empleado | Si |
| Departamento | Estructura organizacional | Si |
| Puesto | Rol laboral | Si |
| Fecha ingreso | Inicio laboral | Si |
| Salario base | Base de calculo de nomina | Si |
| Moneda salario | Moneda del salario | Si |
| Correo | Contacto/login si aplica | Segun caso |
| Estado | Activo o inactivo | Si |
